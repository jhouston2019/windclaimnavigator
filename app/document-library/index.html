How to use

Stay organized

  • Start with the category packet and tailor the lists to the claim.
  • Use the breadcrumbs to navigate back to the main library quickly.
  • Keep exhibits labeled with dates, sources, and claim numbers.

Quality checks

Before sending

  • Verify recipients, deadlines, and delivery methods for each packet.
  • Confirm totals and citations match the attached exhibits.
  • Save a PDF copy with the same naming across the file.
