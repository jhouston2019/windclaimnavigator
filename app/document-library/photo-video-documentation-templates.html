Visuals Consistency Metadata

Overview

Purpose

  • Standardize how photos and videos are captured.
  • Use file naming with location, date, and subject.
  • Include context notes for each set of visuals.

Quality checklist

Ready-to-send

  • Ensure time and date stamping.
  • Capture wide, mid, and detail shots for each area.
  • Back up files immediately to secure storage.

What to include

Use these pieces to keep the photo & video documentation templates packet complete.

Core documents

Build the file

  • Shot list template by room or elevation.
  • File naming and folder structure guide.
  • Photo/video consent or release forms where needed.

Supporting proof

Evidence to attach

  • Sample annotated photos.
  • Equipment and settings checklist.
  • Cloud sharing instructions and permissions.

Delivery notes

Keep it moving

  • Store originals plus compressed copies for sharing.
  • Share via link with view-only access.
  • Maintain metadata; avoid stripping EXIF on export.
