Required Documentation (All Carriers)

  • Policyholder information, claim number, and loss date
  • Comprehensive photo set (pre-mitigation, during, after)
  • Receipts for emergency mitigation and temporary repairs
  • Contractor estimates with scope, pricing, and code references
  • Proof of loss form signed and dated
  • Contents inventory with values and supporting receipts
  • Inspection notes and communication log entries
  • Invoices and permits for completed repairs
